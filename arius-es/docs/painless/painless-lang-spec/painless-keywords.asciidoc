[[painless-keywords]]
=== Keywords

Keywords are reserved tokens for built-in language features.

*Errors*

* If a keyword is used as an <<painless-identifiers, identifier>>.

*Keywords*

[cols="^1,^1,^1,^1,^1"]
|====
| if | else | while | do | for
| in | continue | break | return | new
| try | catch | throw | this | instanceof
|====
