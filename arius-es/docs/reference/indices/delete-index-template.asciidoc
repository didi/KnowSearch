[[indices-delete-template]]
=== Delete index template API
++++
<titleabbrev>Delete index template</titleabbrev>
++++

Deletes an existing index.

////
[source,console]
--------------------------------------------------
PUT _template/template_1
{
 "index_patterns" : ["te*"],
  "settings": {
    "number_of_shards": 1
  }
}
--------------------------------------------------
// TESTSETUP
////

[source,console]
--------------------------------------------------
DELETE /_template/template_1
--------------------------------------------------
